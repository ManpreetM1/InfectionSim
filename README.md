# InfectionSim - CPSC 571
## Introduction
This code simulates how an infection spreads throughout a community, using real-world numbers for the infection parameters. It includes interventions that affect how the infection spreads, they can be turned on through the code and allow you to see how effective these interventions are based off of numbers from peer-reviewed studies.
So far these interventions include:
- Vaccination
- Isolation
- Vaccination + Isolation

## Dependencies
This simulator is fairly lightweight so it only requires:
- Python 3.12 or greater
- Matplotlib
- Pandas

## Installation
Simply download the code and the required dependencies, and it will run in any python interpreter.
